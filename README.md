# LLM_Meditation

A simulation of a meditation session using Google's Gemini API.

## Setup

1.  **Install Dependencies**:
    ```bash
    pip install google-genai
    ```

2.  **API Key**:
    Ensure your API key is set in `Meditation_simulator.py`.

## Running the Simulation

Run the script:
```bash
python Meditation_simulator.py
```

## Output

The simulation logs are saved in the `outputs/` directory as markdown files with a timestamp.
